Typora

简介

  • Typora 是一款简洁、高效与强大的 Markdown 编辑器。而 Markdown 是一种轻量级且易于使用的标记语言，可在 Github/CSDN/简书/印象笔记 等平台上用统一的格式写作。
  • 支持导出为 PDF、HTML，安装 pandoc 之后支持导出为 Word 等格式。
  • 拥有丰富的主题样式可选，可通过 css 自定义。
