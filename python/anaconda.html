序言

Anaconda 是世界上最受欢迎的科学数据平台，扮演者包管理者和环境管理者的角色。

Anaconda Distribution（发行版） 由一个充满活力的开源贡献者社区和全球1800多万用户提供支持，是那些希望在科学计算项目中使用 Python 或 R 的独立数据科学家的首选工具。

Anaconda Distribution（发行版） 有如下特点：

  1. 免费

  2. 易于安装

  3. 包含了

    • Python 发行版
    • 超过 7500 个开源包
    • 免费的社区支持

  4. 与平台无关

Conda 常用命令

安装好 Anaconda 之后，打开 Anaconda Prompt，使用 Conda 命令管理环境与包。

PyCharm 中，File→Settings…→Project→Project Interpreter，可以使用 Conda 环境。

Anaconda 也同时提供 pip 的命令，在 Conda 无法完成某些任务时可选用。
